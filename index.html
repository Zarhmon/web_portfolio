YuBot

YuBot was a group project create for our client in the CITS3200 unit Professional Computing. The bot had a number of features required for acceptance including, maintaing casual conversation, customisable user avatars, provide entertainment through jokes, search with Google and Google Scholar. Overall we managed to implement the required features but due to the timeframe of 10 weeks the scope was limited. In retrospect the dekstop application should have been a web application for ease of user access and tkinter was used for the GUI but had graphical fedality limitations.
